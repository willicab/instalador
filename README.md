Instalador Vivo para distribuciones basadas en Debian
======

## Dependencias
* python-imaging
* lshw
* parted
* reiserfsprogs
* python-parted
* python-webkit
    
```bash
aptitude install python-imaging lshw parted reiserfsprogs python-parted python-webkit
```
